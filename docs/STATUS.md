# 개발 상태

## 구현됨

- Next.js 앱 세팅
- 행동 트래커 첫 화면
- 이메일/비밀번호 로그인 UI
- Supabase 기록 저장 구조
- Supabase 미연결 시 브라우저 임시 저장
- 오늘 기록 화면
- 기록 목록 화면
- 월간 요약 화면
- 식단 점수와 짧은 칭찬 문구
- 독서 쪽수와 진행률 계산
- 모바일/PC 반응형 레이아웃
- Motion 기반 애니메이션

## 아직 안 됨

- 실제 Supabase 프로젝트 생성
- Vercel 배포
- 옵시디언 연동
- 식단 사진/칼로리 입력
- 시쓰기 구글 독스 생성
- 책 상세 화면

## 다음 작업

1. Supabase 프로젝트를 만들고 `supabase/schema.sql`을 실행합니다.
2. `.env.local`에 Supabase 값을 넣습니다.
3. 로그인 후 기록 저장을 확인합니다.
4. Vercel에 배포합니다.

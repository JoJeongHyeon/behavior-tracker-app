# 쉬운 용어 설명

## Next.js

React로 웹사이트를 만들기 쉽게 해주는 틀입니다.

## React

버튼, 입력칸, 카드 같은 화면 조각을 만드는 도구입니다.

## Supabase

로그인과 데이터 저장을 맡아주는 서비스입니다.

## Vercel

만든 웹사이트를 인터넷 주소로 열 수 있게 배포해주는 서비스입니다.

## 환경변수

비밀번호나 서비스 주소처럼 코드에 직접 넣지 않는 설정값입니다.

## DB

기록을 저장하는 데이터베이스입니다.

## RLS

로그인한 사용자가 자기 기록만 볼 수 있게 막는 Supabase 보안 규칙입니다.

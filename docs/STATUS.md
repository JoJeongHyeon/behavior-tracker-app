# 개발 상태

## 현재 상태

행동 트래커 1차 버전은 배포까지 완료되었습니다.

- 로컬 앱: http://localhost:3000
- 배포 앱: https://behavior-tracker-app-khaki.vercel.app
- GitHub: https://github.com/JoJeongHyeon/behavior-tracker-app
- Supabase: https://supabase.com/dashboard/project/otlptxdkaoasfjgfxoys

## 구현됨

- Next.js 앱 세팅
- Supabase 로그인 구조
- 회원가입 진행 상태 UI
- 인증 메일 발송 안내 화면
- 인증 메일 다시 보내기
- 인증 관련 오류 한국어 안내
- Supabase DB 저장 구조
- Vercel 배포
- 오늘 기록 화면
- 기록 목록 화면
- 월간 요약 화면
- 몸무게, 식단, 운동, 공부, 시쓰기, 책읽기, 기타 입력
- `예/아니오` 선택 후 세부 입력 표시
- 식단 점수 계산
- 짧은 칭찬 문구
- 독서 읽은 쪽수와 진행률 계산
- 모바일/PC 반응형 레이아웃
- Motion 기반 애니메이션
- Pretendard 폰트
- 사용자 확인용 문서
- 가입/로그인 계획 문서

## 사용자가 지금 확인할 것

- 배포 주소에서 회원가입이 되는지
- 이메일 인증이 필요한지
- 인증 메일 발송 화면이 명확한지
- 인증 메일 다시 보내기가 작동하는지
- 로그인 후 오늘 기록이 저장되는지
- 기록 목록에 저장한 기록이 보이는지
- 월간 요약 숫자가 맞게 바뀌는지
- 휴대폰에서 입력이 편한지
- 칭찬 문구가 어색하지 않은지
- 애니메이션이 충분하지만 산만하지 않은지

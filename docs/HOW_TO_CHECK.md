# 직접 확인하는 방법

## 실행 확인

- `npm run dev`를 실행합니다.
- `http://localhost:3000`에서 앱이 열리는지 봅니다.
- 샘플 랜딩페이지가 아니라 기록 앱 화면이 바로 보여야 합니다.

## 기록 확인

- 날짜와 기분을 입력합니다.
- 몸무게, 식단, 운동, 공부, 시쓰기, 책읽기를 각각 `예/아니오`로 눌러봅니다.
- `예`를 누른 항목만 입력칸이 열리는지 확인합니다.
- 저장 후 오른쪽 피드백 문구가 바뀌는지 확인합니다.

## 식단 확인

- 간식과 야식을 모두 안 먹음으로 두면 `계속 그렇게 가보자`가 나와야 합니다.
- 간식만 먹고 야식을 안 먹으면 `야식은 잘 넘겼어`가 나와야 합니다.
- 야식만 먹고 간식을 안 먹으면 `간식은 잘 참았어`가 나와야 합니다.

## 독서 확인

- 시작 페이지 10, 끝 페이지 35를 입력하면 26쪽으로 계산되어야 합니다.
- 총 페이지 100을 함께 입력하면 진행률이 35%로 보여야 합니다.

## 동기화 확인

- Supabase 연결 후 같은 계정으로 PC와 모바일에 로그인합니다.
- 한 기기에서 기록한 내용이 다른 기기 기록 목록에도 보이면 성공입니다.

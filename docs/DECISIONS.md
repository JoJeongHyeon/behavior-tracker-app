# 결정 기록

## Next.js

React로 화면을 만들면서 로그인, 배포, 라우팅, 서버 확장까지 자연스럽게 가져가기 위해 선택했습니다.

## Supabase

모바일과 PC에서 같은 기록을 보기 위해 로그인과 데이터베이스가 필요합니다. Supabase는 인증과 DB를 함께 제공해서 1차 버전에 적합합니다.

## Vercel

Next.js 배포에 가장 잘 맞고 무료 공개 주소를 바로 받을 수 있어 선택했습니다.

## Pretendard

한국어 서비스 앱에서 자주 쓰이고 화면이 깔끔하게 보이는 폰트라 선택했습니다.

## 임시 저장 모드

Supabase 환경변수가 없을 때도 UI를 확인할 수 있도록 브라우저 임시 저장 모드를 둡니다. 실제 사용은 Supabase 연결 후를 기준으로 합니다.
